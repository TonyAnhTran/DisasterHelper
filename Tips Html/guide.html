DISASTER HELPER INSTRUCTIONAL GUIDE

 

I.Get started on the Disaster Helper application

  Register

Input all field below:

-  Name field: The name you input must be your real name, if you create account with a nick name, please update to real name in Change profile window it is very important to other people can help you in emergency moment. The name is minimum in 6 character to Maximum is 45 character with special character (+-.@_...).

-  Phone number field: The phone number you input must be your current using phone number, it will be come your account ID and cant be change. The phone number is minimum in 6 number to Maximum is 15 number with only + special character.

-  Password field: The password is minimum in 6 character to Maximum is 45 character with special character (+-.@_...), if you want to change password, please access Change password button in main view.

Then click on Sign up button, the system will response Registration successful.

Click on Login button to back to Login view.

 

  Login

After you complete to register account, you can login with your phone number and password.

If login complete, you will see application main view with any features.

  Forgot password.

 

II.Main view

-  In main view will display all feature in Disaster helper application:

-  Map view: The map feature will help you to see the list and picture of Victim who need your help, you can user map to search place, store, bank, hospital and you can get directions from your location to every where.

-  Contacts: You need to add all phone number can help you in emergency moment, they will help you when you send request help or when application detected you got a accident.

-  Notification: Notification center is look like your message inbox, it will receiver the notification from system about disaster, victims need help near you, who need your help and message from other user.

-  Weather: The current weather of your location will be display on this button, after you click on this button, you can see weather display as map view, and you can search other place weather.

-  Offline tips: Offline tip is collections of experience about disaster, accident, health care, emergency. You can use this feature with out network.

-  Change password: Change your password to new password.

-  Update profile: Update your profile with name and picture.

-  Instructions: The instructions to using this application.

-  Send request help: Is most important feature to save your live when you need help, be careful when click on this button, you only can click it when you need emergency help!

III.User map feature

  Display victims on map and work with victim marker

 

When you load map, it will automatic load current victim list to map and display as marker, picture.

 

When you click on the victim icon, it will display the some options, and you can choose:

-  Victim is safe now: You can choose this option when you was help victim, or other people was help him/her. Please careful when choose this button, it is victims life.

-  Get directions to victim: Get directions from your location to victim location by driving, you can change directions type to walking, motorbike in setting.

-  This use is spammer: When you stay on victim location, and you make sure dont have some one or some accident need your help, you can make this choice.

-  Cancel: Click it to back to map.

 

 

  Change map view: Click on that icon to change map view type from Normal to Hybrid and other.

  Get directions to places

-      Get directions to marker: By click on marker, you can choose get directions to here, it will get direction from your location to marker location.

-      Get directions to victims: By click on victim markers, you can choose get directions to here, it will get direction from your location to marker location.

-      Get directions between to point: you can add to marker by long press hold on map view, it will get directions between that two point you press.

-      Get directions by place, coordinate: click on directions button icon, input start and destination point and click ok.

 

  Search place: By click and input data in search box, you can search any place you want. However, for more correctly result, you should input more clearly data as possible. Example: hospital in Hoa khanh, Danang.

  Get victim: Click on that icon button to update victims list on map.

  Clear Map: Click on that icon button to clear map view data (marker, victims list).
